Структура документа

  • Заголовок.
  • Цель, задачи (для чего предназначен документ, что потребуется для выполнения задачи, настройки).
  • Процедура 1 (последовательность шагов для достижения цели):
    • Шаг 1.1 (законченная операция или действие).
    • Шаг 1.2.
    • Шаг ...
  • Процедура 2:
    • Шаг 2.1.
    • Шаг 2.2.
    • Шаг ...
  • Итог (результат выполнения всех шагов; итог должен совпадать с заявленными целями).
  • Рекомендации (ссылки на похожие задачи).

Структура нормативного акта

  1. Разделы / главы = имеют название и нумеруются: разделы — римскими цифрами с точкой (I., II., III., IV.), главы - арабскими цифрами с точкой (1., 2., 3., 4.).
  2. Статьи = имеют название и нумеруются арабскими цифрами с точкой: 1., 2., 3., 4.
  3. Части статьи = нумеруются арабскими цифрами с точкой (первая цифра — номер статьи, вторая — номер части): 1.1., 1.2., 1.3.
  4. Пункты части статьи = нумеруются арабскими цифрами со скобкой или буквами со скобкой без точки: 1), 2), А), Б)
  5. Абзацы части статьи или пункта = не нумеруются.
  6. Подпункты пункта = нумеруются маленькими буквами со скобкой без точки: а), б), в), г)
