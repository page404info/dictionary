Hello.
This program helps to test the adaptability of your web application in different environments: test, preprod, prod.

STEPS:
1.open index.html.
2.input url to check.
3.click the "Check" button.
4.go to:
- "mobile" - for mobile testing;
- "tablet" - for tablet testing;
- "desktop" - for desktop testing;
- "customer" - for customer size device testing;
- "Bootstrap" - for Bootstrap grid testing;
- "zoom" - for zoom testing (75% - 150%):
	"Ctrl" + "-" - to zoom out;
	"Ctrl" + "+" - to zoom in;
	"Ctrl" + "0" - to restore the default zoom.

!!! If the site is not displayed:

- use "Ignore X-Frame headers" extension for Chrome brouser

Привет.
Эта программа поможет проверить адаптивность Вашего веб-приложения в разном окружении: тест, препрод, прод.

ШАГИ:
1.открыть index.html.
2.ввести адрес сайта для проверки.
3.кликнуть кнопку "Check".
4.перейти в раздел:
- "mobile" - для проверки на мобильном устройстве;
- "tablet" - для проверки на планшете;
- "desktop" - для проверки на ПК;
- "customer" - для проверки на устройстве с заданными параметрами;
- "Bootstrap" - для проверки Bootstrap сетки;
- "zoom" - для проверки масштабируемости (75% - 150%):
	"Ctrl" + "-" - уменьшить;
	"Ctrl" + "+" - увеличить;
	"Ctrl" + "0" - восстановить масштаб по умолчанию.

!!! Если сайт не отображается:

- использовать Chrome расширение "Ignore X-Frame headers"
https://chrome.google.com/webstore/detail/ignore-x-frame-headers/gleekbfjekiniecknbkamfmkohkpodhe
